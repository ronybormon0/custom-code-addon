== Description ==
Custom Code Addon allows you to easily add custom code to your WordPress site. You can easily add CSS, JavaScript, or PHP code.

== Installation ==
1. Download the plugin.
2. Upload it to the wp-content/plugins directory.
3. Activate the plugin from the WordPress dashboard.

== Frequently Asked Questions ==
= How do I add code? =
Go to the Custom Code Addon menu in the dashboard and add your code.
